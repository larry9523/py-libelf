=========
PY-LIBELF
=========

Python bindings for ``libelf`` which is part of elfutils, https://sourceware.org/elfutils/

Install
*******

.. code-block:: bash

        cd build
        cmake ../
        make


Test
****

.. code-block:: bash

        cd build
        cmake ../
        make install
        make test
